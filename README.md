## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Cody-Le/NbodyPy.git
   ```
2. Edit any Variable to your liking
   ```sh
   height, width = [30,30]
   underPopRate = 2
   highPopRate = 3
   minimumSpawnRate = 3
   ```


2. Run main.py
   ```sh
   py main.py
